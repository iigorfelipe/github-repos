<h1 align="center">Search github users</h1>

<p align="center">
  <i>
    Search for a github user and see all your public repositories in a list, grid or carousel.
  </i>
</p>

## Technologies

- TypeScript
- React - `Vite (react-ts)`
- Styled Components

## Development Tools

- VScode

## Install

```bash
git clone git@github.com:iigorfelipe/github-repos.git
```

```bash
cd repos-github
```

```
npm install
```

## Local Usage

```bash
npm run dev
```
